# 🕸️ Topic Web — Cross-Industry Talking Points

Type **two industries** and Topic Web builds an interactive web of content topics that *both* can authentically talk about — color-coded by which industry each leans toward, sized by opportunity, and expandable into layers of sub-topics.

**Live demo:** _(GitHub Pages link appears here after publishing)_

## How to use
1. Type **Industry A** (left) and **Industry B** (right) — free-form, anything.
2. Set **topics** (base count) and **layers** (depth — each topic branches into more specific sub-topics).
3. Hit **⚡ Generate**. Click any orb for a full brief: the talking point, why it works for each side, and questions each industry would ask the other.
4. Scroll to **zoom**, drag the background to **pan**, click an orb to lock it center, **⤢ Fit** to frame everything.

No API key needed to explore — hit **Sample** for an instant demo. Live generation uses free AI models via OpenRouter (free-only by default, so it costs nothing).

## Reading the web
- **Red** orb = leans to Industry A · **Blue** = leans to Industry B · **Purple** = balanced.
- **Darker / bigger** orb = stronger content opportunity.
- **Dashed amber lines** connect related topics across different branches.

## Notes
- It grounds the AI in real current chatter from **Hacker News + Wikipedia** (free, no key).
- Free AI models are a shared pool and get busy at peak times, so a run may need a retry. The **📋 Log** button shows exactly what happened.
- Settings (**⚙**): pick a default model, toggle "Free models only", or paste your own OpenRouter key.

Single self-contained `index.html` — no build step, no dependencies.
